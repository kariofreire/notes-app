### :rocket: Tecnologias Utilizadas
- [Node Js](https://nodejs.org/en/)
- [React Native](https://reactnative.dev/)
- [Expo](https://expo.io/)

### :pencil: funcionalidades
- [ ] Criação de tarefas
- [ ] Listagem de tarefas
- [ ] Atualização de tarefas
- [ ] Exclusão de tarefas

### :wrench: Uso:
```bash
yarn install # install dependencies
# or npm install

yarn start # run app
# or npm start
```
